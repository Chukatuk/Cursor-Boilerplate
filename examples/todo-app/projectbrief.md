## Project Name

TaskFlow

## Core Vision

A minimal, fast todo app with keyboard-first UX. Built as a single-page web app with offline support.

## Target Audience

Developers and power users who want a distraction-free task manager that works offline and syncs when connected.

## Primary Objectives

- [ ] Keyboard-driven task management (create, complete, delete without touching the mouse)
- [ ] Offline-first with local storage, sync to server when online
- [ ] Sub-200ms UI response time for all interactions

## Scope

### In scope
- Task CRUD (create, read, update, delete)
- Keyboard shortcuts for all actions
- Offline storage with IndexedDB
- Server sync via REST API
- Dark mode

### Out of scope
- Team collaboration / shared lists
- Mobile native app (web-only for now)
- Calendar integration
- Recurring tasks

## Success Criteria

- All core flows work without a mouse
- App works fully offline, syncs without data loss when reconnected
- Lighthouse performance score > 95
